# Google Stitch — Mockup Prompts

Use these prompts in [Google Stitch](https://stitch.withgoogle.com/) to generate UI mockups for each section of the portfolio site. Run each prompt separately to get focused mockups, then assemble the full-page composition.

**Design language:** Modern editorial, smart & clean. Deep blue + sage on cream. Serif headings (Playfair Display), sans-serif body (Inter).

---

## Prompt 1 — Full Page Overview

```
Design a single-page portfolio website for a frontend engineer in a modern editorial style.

The page has these sections stacked vertically: sticky navigation bar (name wordmark left, links right), full-height hero with interactive feel, about me with two-column text and photo layout, skills and experience section with integrated timeline, projects grid with 3-4 case study cards, contact section with scheduling CTA, inline resume section with embedded PDF, and a minimal footer.

Style: smart neutral color palette — off-white cream background (#FAFAF5), near-black text (#1A1A1A), deep matte blue accent (#1E3A5F), muted sage secondary (#8B9D8B). Typography uses serif headings (like Playfair Display) with sans-serif body text (like Inter). Asymmetric editorial grid layouts with generous whitespace. Minimal UI chrome — no heavy borders or shadows.

Desktop viewport, 1440px wide.
```

## Prompt 2 — Navigation Bar

```
Design a sticky navigation bar for a portfolio website in modern editorial style.

Left: person's name in serif font as wordmark.
Right: anchor links — "About", "Experience", "Projects", "Contact", "Resume" — in clean sans-serif. A "View Resumé" button styled as a CTA with deep blue (#1E3A5F) accent — either filled or outlined.

Background: off-white cream (#FAFAF5) with a subtle bottom border or shadow on scroll. Typography is refined and minimal. Desktop width 1440px. Also show a mobile version with hamburger menu.
```

## Prompt 3 — Hero Section

```
Design a hero section for a frontend engineer portfolio website in modern editorial style.

Full viewport height. Asymmetric layout. Left side has a large serif headline with the person's name, "FRONTEND ENGINEER & UI CRAFTSMAN" subtitle in small caps sans-serif, and a 1-2 line elevator pitch below. Two CTA buttons: "View Work" (filled in deep blue) and "View Resumé" (outlined).

Right side has overlapping abstract geometric shapes (circles, organic blobs) in deep navy blue (#1E3A5F) and muted sage (#8B9D8B). Within the shapes, include a small code snippet element and a color palette/swatch element suggesting the intersection of engineering and design. Subtle atmospheric quality — slight transparency, depth, blur effects. A scroll-down indicator at the bottom.

Colors: off-white cream background (#FAFAF5), near-black text (#1A1A1A), deep blue accent (#1E3A5F), muted sage (#8B9D8B). Typography: serif for headline, clean sans-serif for everything else. Desktop 1440px. Modern, atmospheric, smart, and clean.
```

## Prompt 4 — About Me Section

```
Design an "About Me" section for a portfolio website in modern editorial layout.

Two-column split: left column has 2-3 paragraphs of bio text in sans-serif body font with a serif section heading "About" at the top. Right column has a professional portrait photo with editorial cropping — slightly asymmetric placement, maybe with a muted tinted overlay or offset border in sage (#8B9D8B).

Include a pull quote in large serif italic text between the paragraphs — something like "I believe great interfaces are invisible." Generous whitespace, cream background (#FAFAF5), near-black text (#1A1A1A), deep blue (#1E3A5F) for accent details. Desktop 1440px wide.
```

---

## Prompt 5 — Skills & Experience Section

```
Design a skills and experience section for a portfolio website in modern editorial style, where tech skills and work history are integrated into one cohesive layout.

Left portion: tech stack grouped by category — "Languages", "Frameworks", "Tools" — displayed as clean typographic lists or minimal tag groups. No progress bars or rating systems.

Right portion: compact work experience timeline — each entry has company name in bold, role title, date range, and a 1-line description. Connected by a subtle vertical line or timeline marker in deep blue (#1E3A5F).

Section heading "Skills & Experience" in serif. Clean sans-serif for all list content. Cream background (#FAFAF5), near-black text (#1A1A1A), muted sage (#8B9D8B) for secondary text. Desktop 1440px, two or three column grid.
```

---

## Prompt 6 — Projects / Case Studies Section

```
Design a projects section for a frontend engineer portfolio in modern editorial grid style.

Section heading "Selected Work" in serif font. Below it, a grid of 3-4 project cards with varying sizes for visual hierarchy — one card larger than the others. Each card has: a project screenshot/thumbnail, project title in serif, a short 1-line description, and small tech tags (React, TypeScript, etc.) in monospace.

Show one card in an expanded/inline-expanded state below the grid: it reveals a full case study with sections for Context, Approach, Outcome, and Stack, with a larger screenshot and links to live site and GitHub.

Colors: cream background (#FAFAF5), cards with very subtle warm gray (#EDEAE5) backgrounds, deep blue (#1E3A5F) accent on hover states and links, muted sage (#8B9D8B) for secondary text. Clean, typographic, magazine feel. Desktop 1440px.
```

## Prompt 7 — Contact Section

```
Design a contact section for a portfolio website in modern editorial style.

Centered layout with generous whitespace. Large serif heading "Let's Work Together". Below it, a prominent "Book a Call" button styled in deep blue (#1E3A5F) linking to Calendly. Below that, an email address as a mailto link. Below that, a row of two social icons — GitHub and LinkedIn — in near-black, minimal style.

Cream background (#FAFAF5), near-black text (#1A1A1A), muted sage (#8B9D8B) for secondary text. Desktop 1440px.
```

## Prompt 8 — Resume Section + Footer

```
Design an inline resume section and footer for a portfolio website in modern editorial style.

Resume section: serif heading "Resumé" at top. Below it, a single-page resume PDF displayed as an embedded document or high-quality image, centered on the page with subtle shadow or border. A "Download PDF" button in deep blue (#1E3A5F) positioned to the right of the heading or below the resume. The resume should be readable at this size.

Footer: simple, with a thin top border in muted sage (#8B9D8B). Contains copyright text, repeated social links (GitHub, LinkedIn icons), and optional "Built with Next.js" credit. All in small sans-serif text, muted sage color.

Cream background (#FAFAF5). Desktop 1440px.
```

## Prompt 9 — Mobile Layout

```
Design a mobile version (375px wide) of a frontend engineer portfolio website in modern editorial style.

Show the full page stacked vertically: hamburger navigation at top with the name as wordmark, hero section with large serif heading and tagline (no cursor effects, static geometric shapes), about section with photo above text in single column, skills as a simple list with experience timeline below, project cards stacked vertically with one expanded, contact section centered with booking button, inline resume with download button, and a simple footer.

Cream background (#FAFAF5), near-black text (#1A1A1A), deep blue accent (#1E3A5F), muted sage secondary (#8B9D8B). Serif headings, sans-serif body. Touch-friendly spacing and tap targets.
```

---

## Tips for Stitch

- Run each prompt individually for best results
- If a section doesn't match your vision, iterate by adding more specific details to the prompt
- Once you have mockups for each section, you can create a composite or use them as reference for implementation
- Export mockups as images to include in the project for developer reference
- Colors and copy in mockups are approximate — the PRD is the source of truth
