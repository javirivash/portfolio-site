# javirivash.dev

Hello! This is my personal portfolio site.

## Built With

- **Next.js 16** — App Router, server components, Turbopack
- **Tailwind CSS v4** — CSS-based design tokens via `@theme`
- **Framer Motion** — Cursor-driven parallax, scroll animations, layout transitions
- **TypeScript** — Strict mode throughout

## Features

- Magazine/editorial design with mixed serif + sans-serif typography
- Interactive hero with cursor-driven parallax shapes
- Inline expandable project case studies
- Embedded PDF resume viewer
- Fully responsive (mobile, tablet, desktop)
- Accessible — semantic HTML, reduced-motion support

## Running Locally

```bash
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000).
